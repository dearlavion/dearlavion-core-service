# dearlavion-core-service

Spring Boot (Java 21) service with MongoDB, ModelMapper, and Swagger (springdoc).
Base package: com.dearlavion.coreservice

Run:
1. Ensure MongoDB is running at the URI in src/main/resources/application.properties
2. mvn clean package
3. java -jar target/dearlavion-core-service-0.0.1-SNAPSHOT.jar
Swagger UI: http://localhost:8080/swagger-ui/index.html
